Weather Time Directions Agent - Project Description

Project Overview

The Weather Time Directions Agent is a production-ready Streamlit application featuring an intelligent AI agent that provides real-time weather information, accurate time data, and directions between cities using Google's Gemini AI and external APIs.

Streamlit Cloud (Recommended):

  • Automatic scaling and hosting
  • Built-in secrets management

Configuration Requirements

Required API Keys:

  • Google AI API key (for Gemini LLM)
  • Google Maps Platform key (for directions)
  • Opik API key (for monitoring)
  • OpenWeatherMap key (for weather data)
  • TimeZoneDB key (for timezone data)
Environment Setup:
Copy .env.example to .env and fill in your API keys.
